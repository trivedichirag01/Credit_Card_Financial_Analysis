# Credit Card Weekly Financial Dashboard

## Overview
This project analyzes weekly credit card performance and presents key financial insights through an interactive Power BI dashboard.  
The workflow covers data preparation in Excel, transformation in SQL, and visualization in Power BI.

---

## Tools Used
- **Excel** – Data cleaning and preparation  
- **SQL** – Data transformation and aggregation  
- **Power BI** – Data modeling, DAX, and dashboard development  

---

## Workflow

Raw Data → Excel Cleaning → SQL Transformations → Power BI Data Model → Dashboard

---

## Key Metrics & Analysis

- Weekly revenue and growth trends  
- Transaction volume and value  
- Customer segmentation by age, income, and card category  
- Regional and merchant performance  
- Delinquency and risk indicators  

---

## Dashboard Capabilities

- Weekly performance tracking  
- Interactive filtering and drill-downs  
- Customer behavior analysis  
- Revenue and risk monitoring
